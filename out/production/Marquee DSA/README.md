# Marquee 2028 - DSA Practice

Welcome to the **Marquee 2028** repository! This project contains various Data Structures and Algorithms (DSA) programs and practice problems implemented in **Java**.

## 🚀 Contents

Currently, this repository includes several foundational Java programs:

- **Fibonacci Series** (`fibonacci.java`)
- **Greatest Common Divisor** (`GCD.java`)
- **Pattern Printing** (`pattern.java`, `pattern2.java`)
- **Basic Calculators & Flow Control** (`1.java`, `sample.java`, `gac.java`)

## 🛠️ Prerequisites

To run these programs, you'll need the Java Development Kit (JDK) installed on your system.

- [Download JDK](https://www.oracle.com/java/technologies/downloads/) (Java 11 or higher recommended, tested on Java 21)

## 💻 How to Compile and Run

You can run any of these files directly from your terminal.

1. **Open your terminal** and navigate to the project directory:
   ```bash
   cd dsa
   ```

2. **Compile the Java file** (Replace `Filename.java` with the file you want to run):
   ```bash
   javac Filename.java
   ```

3. **Run the compiled Java class**:
   ```bash
   java Filename
   ```

*(Alternatively, for Java 11 and above, you can run single-file programs directly without manual compilation: `java Filename.java`)*
